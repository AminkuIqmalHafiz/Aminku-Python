# Aminku_Python
The Rule: Zero AI Assistance
This repository is my personal python workbench for foundational computer science and quantitative economics. No Large Language Models (LLMs) or AI auto-complete tools were used to generate the code in this directory.

Objective
While my Aminku-Financial-Engineering repositories focus on staistical system framework, this space is dedicated to raw, unassisted progression. It is designed to connect the gap between high-level quantitative logic and low-level programming fundamentals. If a script takes five hours to debug manually, it takes five hours.

Current Focus Areas:
Understanding Python Syntax : Understanding syntax, built-in functions, library functions and connect it with mathematics and logic.
Quantitative Economics : Translating heavy mathematical theory, Markov chains, and dynamic programming into raw Python.
Data Structures & Algorithms: Mastering the underlying mechanics required to optimize execution speed and technical skill.
Object-Oriented Programming (OOP): Building code environments from scratch.
